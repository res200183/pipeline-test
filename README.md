# AWS CI/CD Pipeline Demo

This project demonstrates a simple CI/CD pipeline using AWS services.

## Architecture

GitHub → AWS CodePipeline → AWS CodeBuild

## How it works

1. Developer pushes code to GitHub
2. GitHub triggers AWS CodePipeline
3. CodePipeline starts AWS CodeBuild
4. CodeBuild executes commands from buildspec.yml

## Technologies

- Git
- GitHub
- AWS CodePipeline
- AWS CodeBuild
- YAML

## Purpose

This repository was created as a hands-on practice project to demonstrate CI/CD pipeline implementation in AWS.
